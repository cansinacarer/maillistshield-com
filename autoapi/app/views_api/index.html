app.views_api.validate_single()[source]

The API endpoint to validate a single email address.

This endpoint requires the request content-type to be application/json and a JSON body with an “email” key.

Optionally, the request JSON can include a boolean key “single_level_response”. If set to true, the API will return the validation result in a single-level dictionary. Otherwise, and by default, the response includes status and message keys.
